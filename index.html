  • Blood Sugar Prediction

    Machine Learning

    (50.039) Theory and Practice of Deep Learning - 1D Project This project "Blood Sugar Prediction" uses glucose monitor, heart rate, insulin and carbohydrate intake, total steps walked, and calories burnt—aggregated into five-minute intervals over a sliding six-hour window—to predict glucose levels one hour ahead (bg+1:00). Missing glucose and heart-rate readings are linearly interpolated (with row-mean fallback), while insulin, carbohydrate, steps, and calorie gaps are zero-filled and any residual NaNs forward-then backward-filled. For training, small Gaussian noise (1% of each feature’s standard deviation) augments the dataset, and only the most recent 24 time steps of each of the six sensor channels (i.e., two hours of data) are retained. Finally, all features and the target glucose value are standardized to zero mean and unit variance. A two-block Temporal Convolutional Network then processes each sample as a 24×6 sequence: each block applies dilated 1D convolutions with chomp layers and residual connections to extract temporal features, ending in a linear layer that outputs a single glucose prediction. The model is trained with MSE loss, Adam optimization, and early stopping, using an 80/10/10 split for training, validation, and testing. Predictions are evaluated with RMSE, MAE, MARD, MBE, MAPE and Pearson’s r.

  • Movie Recommender System

    Machine Learning

    This project presents a movie recommender system built on the MovieLens dataset that delivers personalized movie suggestions by leveraging user ratings, movie metadata, and genre information. It employs neural collaborative filtering, where both users and movies are represented as embeddings in a latent space that capture intrinsic characteristics and preferences. The system utilizes a multilayer perceptron to learn complex interactions between these embeddings, thereby enabling the recommendation engine to suggest movies that a user is likely to enjoy. Additionally, the project offers multiple recommendation strategies: it provides user-based recommendations that identify unseen movies for each user, similarity-based recommendations that use cosine similarity to find movies with closely aligned embeddings, genre-based filtering that targets movies within specific genres, and a combined approach that integrates both similarity and genre criteria for more refined suggestions.
