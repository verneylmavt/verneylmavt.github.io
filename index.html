  • Movie Recommender System

    Machine Learning

    This project presents a movie recommender system built on the MovieLens dataset that delivers personalized movie suggestions by leveraging user ratings, movie metadata, and genre information. It employs neural collaborative filtering, where both users and movies are represented as embeddings in a latent space that capture intrinsic characteristics and preferences. The system utilizes a multilayer perceptron to learn complex interactions between these embeddings, thereby enabling the recommendation engine to suggest movies that a user is likely to enjoy. Additionally, the project offers multiple recommendation strategies: it provides user-based recommendations that identify unseen movies for each user, similarity-based recommendations that use cosine similarity to find movies with closely aligned embeddings, genre-based filtering that targets movies within specific genres, and a combined approach that integrates both similarity and genre criteria for more refined suggestions.
